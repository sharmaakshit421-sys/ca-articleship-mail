# CA Articleship — Bulk Mailer

Tools to send and track personalised articleship applications directly from your Gmail.

### 🚀 Live Tools

1. **[CA Bulk Mailer](https://sharmaakshit421-sys.github.io/ca-articleship-mail/ca-mailer-v3.html)** — Compose and send personalised emails to multiple firms. Import recipients from Excel or add manually.
2. **[Application Tracker](https://sharmaakshit421-sys.github.io/ca-articleship-mail/ca-tracker.html)** — Track sent applications, reply status, and add notes.

---

### ✨ Features

- **Bulk sending** via your own Gmail (Google Apps Script backend)
- **Excel import** — upload `.xlsx`/`.csv` with Name, Firm, Email columns
- **Manual entry** — add recipients one by one
- **CV attachment** — attach PDF/DOC to every email
- **Live email preview** — see exactly how each email looks before sending
- **Application tracker** — auto-synced, with reply status and notes
- **CSV export** — download your full tracker data

---

### 📋 How to Use

1. Open the **Bulk Mailer** link above.
2. Click **⚙ Config** and paste your Google Apps Script Web App URL.
3. Add recipients via **Excel import** or **manual entry**.
4. Write your email, attach your CV, and hit **🚀 Send All**.
5. View sent applications in the **Tracker**.
